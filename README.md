# Жарава · Live Planner

Личен 180-дневен планер (1 юли – 31 декември 2026): хранене, тренировки, навици, контент план и проследяване на прогреса. Работи изцяло в браузъра — без сървър, без акаунти, без external бази данни.

## Как да го пуснеш в GitHub Pages (безплатно, за ~2 минути)

1. Създай нов репозиторий в GitHub (например `zharava`). Може да е **Private** — GitHub Pages работи и за частни репозитории при платените планове; за безплатен план направи го Public.
2. Качи всички файлове от тази папка в репозитория (запази структурата на папките):
   ```
   index.html
   manifest.webmanifest
   sw.js
   icon.svg
   css/style.css
   js/data.js
   js/app.js
   ```
3. В репозитория: **Settings → Pages → Source: Deploy from a branch → Branch: main / (root) → Save**.
4. След минута приложението е живо на `https://<твоят-потребител>.github.io/zharava/`.

### Инсталиране като приложение на телефона

Отвори адреса в Chrome/Safari на телефона → меню → **Add to Home Screen / Добави към началния екран**. Отваря се на цял екран като истинско приложение и работи офлайн.

## Къде се пазят данните

Всички отметки, рефлексии и задачи се пазят **локално на устройството** (localStorage). Нищо не се изпраща никъде.

- **Резервно копие**: таб „Прогрес" → **Експорт (JSON)** — сваля файл с всичко.
- **Прехвърляне на друго устройство**: там отвори „Прогрес" → **Импорт** и избери файла.

> Забележка: данните са отделни за всеки браузър/устройство. Ако ползваш и телефон, и лаптоп — прехвърляй с Експорт/Импорт.

## Структура на кода

| Файл | Какво съдържа |
|---|---|
| `index.html` | Само скелетът на страницата (~2 KB) |
| `css/style.css` | Всички стилове |
| `js/data.js` | Менюта, тренировъчна програма, контент план, навици — **редактирай тук**, ако искаш да смениш ястие или упражнение |
| `js/app.js` | Логиката на приложението (vanilla JS, без библиотеки) |
| `sw.js` | Service worker — кешира всичко за мигновено отваряне и офлайн |
| `manifest.webmanifest` | PWA манифест (инсталиране на телефон) |

## Какво е подобрено спрямо старата версия

- **Мигновено отваряне**: премахнати са React (~140 KB) и Firebase (~300 KB), които се теглеха от CDN при всяко отваряне. Сега приложението е чист JavaScript без нито една външна библиотека и след първото посещение се отваря офлайн от кеша.
- **Без настройки**: старата версия изискваше Firebase акаунт и конфигурация. Сега — качваш файловете и работи.
- **Нов таб „Прогрес“**: карта на всичките 180 дни (heatmap), серии (streaks), седмични графики, статистика, средно изпълнение.
- **Резервно копие**: експорт/импорт на данните като JSON файл.
- **PWA**: инсталира се на телефона като приложение, работи офлайн.
- **Структуриран код**: данни, стилове и логика са в отделни файлове — лесно се редактират.
